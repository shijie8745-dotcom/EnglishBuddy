# EnglishBuddy 需求文档

## 1. 项目概述

EnglishBuddy 是一款面向低龄英语学习者的 iOS 应用，通过与 AI 对话伙伴 Amy 进行互动式学习，提升英语口语能力。应用采用游戏化设计，结合宠物养成和签到奖励机制，激发儿童的学习兴趣。

## 2. 核心功能需求

### 2.1 课程学习系统
- **课程展示**：以列表形式展示英语课程，每课包含标题、副标题和状态图标
- **课程状态跟踪**：
  - 已完成（绿色勾选图标）
  - 学习中（橙色播放图标）
  - 未解锁（灰色锁定图标）- 当前版本所有课程均可访问
- **课程详情**：展示课程介绍、词汇表和句型学习
- **练一练功能**：
  - 快速进入当前选定课程的 AI 对话
  - 可在设置中切换练一练课程
  - 首页卡片显示当前正在学习的课程

### 2.2 AI 对话系统
- **主对话页面（ChatView）**：
  - 语音输入：按住麦克风按钮说话，松开后自动发送
  - AI 多轮对话：支持上下文记忆，实现连续对话
  - 实时语音播放：AI 回复使用 TTS 语音播报
  - 点击消息重播：再次点击 AI 消息可重新播放
- **AI 测试页面（AIChatTestView）**：
  - 手动文本输入：通过键盘输入文字与 AI 对话
  - 系统 TTS：使用 iOS 系统语音播放（节省 API 费用）
  - 页面自动触发：进入页面时 AI 自动发起问候
  - 支持多轮对话：记住上下文，实现连续性对话
- **课程感知**：AI 基于当前课程的主题和词汇进行针对性对话

### 2.3 学习宠物系统（xixi）
- **宠物展示**：
  - 可拖动的小兔子宠物显示在课程页面右下角
  - 显示名称、等级、持有胡萝卜数量
- **喂食机制**：
  - 消耗胡萝卜喂食，每次喂食 +50 经验值
  - 每累积 100 经验值升 1 级
  - 升级阈值：等级 × 100 经验值
- **位置记忆**：记录宠物被拖动的位置，下次打开保持原位
- **统计展示**：在设置页面显示宠物等级、经验、喂食次数等数据

### 2.4 签到系统
- **日历式签到**：
  - 以月历形式展示签到记录
  - 已签到日期显示绿色背景
  - 今日未签到显示橙色边框
- **奖励规则**：
  - 每日签到：+5 胡萝卜
  - 连续 3 天：额外 +5 胡萝卜
  - 连续 7 天：额外 +10 胡萝卜
- **统计数据**：展示累计签到次数、连续签到天数、获得胡萝卜总数

### 2.5 用户系统
- **个人设置**：
  - 用户昵称修改
  - 头像上传（使用系统相册）
  - API Key 设置（用于 AI 对话）
  - AI 模型选择（Qwen2.5/DeepSeek 系列）
  - 语音语速调节
- **学习统计**：
  - 已完成课程数
  - 学习总时长（分钟）
  - 学习会话次数

### 2.6 数据持久化
- UserDefaults 存储：
  - 用户信息（User）
  - 宠物数据（Pet）
  - 学习进度（LessonProgress）
  - 签到记录（CheckInRecord 数组）

## 3. 技术需求

### 3.1 前端技术
- **框架**：SwiftUI（iOS 26+）
- **状态管理**：@Observable 宏
- **导航**：NavigationStack + NavigationLink
- **动画**：SwiftUI 原生动画支持

### 3.2 AI 服务
- **API 兼容**：OpenAI 兼容格式
- **服务提供商**：阿里云 DashScope
- **默认模型**：qwen2.5-7b-instruct（免费）
- **多轮对话**：支持 messages 数组传递历史记录
- **TTS 服务**：阿里云 qwen3-TTS-Instruct-Flash

### 3.3 语音识别
- **框架**：Speech Framework
- **功能**：实时语音转文字
- **权限**：麦克风权限请求

## 4. UI/UX 规范

### 4.1 色彩规范
- **主色调**：
  - 橙色 `#F97316` - 主要按钮、强调色
  - 浅橙色 `#FB923C` - 渐变、次要强调
  - 背景浅色 `#FEF7ED` - 页面背景
- **功能色**：
  - 练一练青色：`#0d9488` → `#2dd4bf`
  - 签到绿色：`#22c55e` → `#16a34a`
  - 蓝色：`#3B82F6` - 用户消息气泡
- **中性色**：
  - 深灰：`#1F2937` - 主要文字
  - 中灰：`#6B7280` - 次要文字
  - 浅灰：`#F3F4F6` - 输入框背景

### 4.2 字体规范
- 标题：系统字体，加粗，28-36pt
- 正文：系统字体，常规，16-18pt
- 辅助文字：系统字体，常规，12-14pt

### 4.3 组件规范
- **卡片**：圆角 16pt，白色背景，轻微阴影
- **按钮**：圆角 12-20pt，橙色主色调
- **头像**：圆形，2pt 边框
