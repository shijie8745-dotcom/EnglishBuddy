# EnglishBuddy 需求文档

## 1. 项目概述

EnglishBuddy 是一款面向低龄英语学习者的 iOS 应用，通过与 AI 对话伙伴 Amy 进行互动式学习，提升英语口语能力。应用采用游戏化设计，结合宠物养成和云朵币奖励机制，激发儿童的学习兴趣。

**支持设备**：iPhone 和 iPad（自适应布局）

## 2. 核心功能需求

### 2.1 课程学习系统
- **课程展示**：以列表形式展示 9 个单元英语课程，每课包含标题、副标题和状态图标
- **课程状态跟踪**：
  - 已完成（绿色勾选图标）
  - 学习中（橙色播放图标）
  - 所有课程均可访问
- **课程详情**：每个单元独立详情页，展示课程介绍、词汇表和句型学习
- **练一练功能**：
  - 快速进入当前选定课程的 AI 对话
  - 可在设置中切换练一练课程
  - 首页卡片显示当前正在学习的课程

### 2.2 AI 对话系统
- **语音对话页面（ChatView）**：
  - 语音输入：按住麦克风按钮说话，松开后自动发送
  - AI 多轮对话：支持上下文记忆，实现连续对话
  - 实时语音播放：AI 回复使用阿里云 TTS 语音播报
  - 点击消息重播：再次点击消息可重新播放
  - AI 教师头像：Amy 形象展示
- **课程感知**：AI 基于当前课程的主题和词汇进行针对性对话

### 2.3 宠物养成系统
- **宠物商店**：
  - 10 款可选宠物形象（音战、酷宝、电喵、赛博、粉喵、酷少、银刃、绿萝、绯红、机娘）
  - 每只宠物售价 200 云朵币
  - 宠物预览功能
- **宠物切换**：已解锁宠物可自由切换
- **浮动展示**：选中的宠物显示在首页，支持拖动定位
- **位置记忆**：记录宠物被拖动的位置，下次打开保持原位

### 2.4 云朵币系统
- **获取方式**：
  - 学习时长：每 1 分钟 = 1 云朵币
  - 每日打卡：对话 10 次 = 5 云朵币
  - 连续 3 天打卡：额外 +5 云朵币
  - 连续 7 天打卡：额外 +10 云朵币
- **消费方式**：
  - 购买宠物：200 云朵币/只
- **打卡机制**：
  - 每日对话 10 次自动触发打卡
  - 日历式展示打卡记录
- **统计数据**：
  - 首页展示：累计打卡 / 学习时长（分钟）/ 累计对话
  - 商店页展示：累计打卡 / 连续打卡 / 今日对话

### 2.5 用户系统
- **个人设置**：
  - 用户昵称修改
  - 头像上传（使用系统相册）
  - API Key 设置（用于 AI 对话）
  - AI 模型选择（Qwen2.5/DeepSeek 系列）
  - 语音语速调节
- **练一练课程选择**：网格布局选择默认练习课程

### 2.6 数据持久化
- UserDefaults 存储：
  - 用户信息（User）
  - 宠物收藏（PetCollection）
  - 学习进度（LessonProgress）
  - 云朵币系统（CloudCoinSystem）
  - 打卡记录（CheckInRecord 数组）

## 3. 技术需求

### 3.1 前端技术
- **框架**：SwiftUI（iOS 26+）
- **状态管理**：@Observable 宏
- **导航**：NavigationStack + NavigationLink
- **动画**：SwiftUI 原生动画支持
- **自适应布局**：Size Class 响应式设计

### 3.2 AI 服务
- **API 兼容**：OpenAI 兼容格式
- **服务提供商**：阿里云 DashScope
- **默认模型**：qwen2.5-7b-instruct（免费）
- **多轮对话**：支持 messages 数组传递历史记录
- **TTS 服务**：阿里云 qwen3-TTS-Instruct-Flash

### 3.3 语音识别
- **服务**：阿里云 ASR（实时语音识别）
- **WebSocket 连接**：支持实时流式识别
- **权限**：麦克风权限请求

### 3.4 资源管理
- **图片存储**：Assets.xcassets（支持 App Thinning）
- **自适应分辨率**：支持 @1x/@2x/@3x

## 4. UI/UX 规范

### 4.1 色彩规范
- **主色调**：
  - 橙色 `#F97316` - 主要按钮、强调色
  - 浅橙色 `#FB923C` - 渐变、次要强调
  - 背景浅色 `#FEF7ED` - 页面背景
- **功能色**：
  - 练一练青色：`#0d9488` → `#2dd4bf`
  - 打卡绿色：`#22c55e` → `#16a34a`
  - 蓝色：`#3B82F6` - 用户消息气泡
- **中性色**：
  - 深灰：`#1F2937` - 主要文字
  - 中灰：`#6B7280` - 次要文字
  - 浅灰：`#F3F4F6` - 输入框背景

### 4.2 字体规范
- 标题：系统字体，加粗，28-36pt
- 正文：系统字体，常规，16-18pt
- 辅助文字：系统字体，常规，12-14pt

### 4.3 组件规范
- **卡片**：圆角 16pt，白色背景，轻微阴影
- **按钮**：圆角 12-28pt，橙色主色调
- **头像**：圆形，2pt 边框

### 4.4 响应式布局
- **iPhone 竖屏**（.compact）：紧凑布局，2-3列网格
- **iPhone 横屏 / iPad**（.regular）：宽屏布局，3-4列网格
- **宠物尺寸**：iPhone 140pt / iPad 250pt
- **图标尺寸**：自适应缩放
